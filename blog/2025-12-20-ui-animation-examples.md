LVGL supports a wide variety of animation effects, providing developers with powerful tools to create smooth and engaging user interfaces. In the early stages of DejaOS development, we focused primarily on functional implementation. Now that we have a solid foundation, we're turning our attention to enhancing the user experience with better visual effects.

<!--truncate-->

## From Functionality to User Experience

During the initial development phase, DejaOS prioritized core functionality and stability. As the platform matured, we recognized the importance of providing developers with rich animation examples and best practices to help them create more polished and professional applications.

To support this goal, we've created multiple animation examples that demonstrate different approaches to implementing smooth animations in DejaOS applications. These examples serve as reference implementations for developers looking to enhance their UI with engaging animations.

- **Example 1 & 3**: These animations are implemented using LVGL's native animation functions, leveraging the framework's built-in animation capabilities for optimal performance.

- **Example 2**: This animation uses GIF image playback with pause and resume functionality, providing developers with an alternative approach for implementing animations.
