# DejaOS 2.1.5 (Latest Recommended)

**DejaOS 2.1.5** is based on **2.1.0** and is the currently recommended release for the [System Manager App](/docs/basics/managerapp) experience.

Main updates:

- **Clearer exit UI**: After an app exits, the device shows a clear status page instead of the last app screen, so users do not mistake the app as still running.
- **Passive entry**: If the main app keeps crashing and the watchdog detects **6 consecutive restarts**, circuit-breaker protection kicks in—the device stays in System Manager so it remains reachable for diagnosis.
- **Production log download**: Download application runtime logs from the Web console under **Maintenance & Logs** (useful when live VS Code logs are unavailable in production).

For details and step-by-step screenshots, see [System Management App](/docs/basics/managerapp).
