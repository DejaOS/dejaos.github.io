# 设备型号选择 (Device Model)

在使用 VSCode 进行 DejaOS 开发时，您需要在项目配置文件 (`app.dxproj`) 中选择正确的 **设备型号 (Device Model)**。

![设备型号选择](/img/device-model-choose.png)

由于列表中包含了许多为了兼容旧版本而保留的型号，初次使用的开发者可能会感到困惑。为了帮助您做出正确的选择，本文档将澄清几个关键概念并提供详细的映射关系。

## 三个“型号”概念的区别

为了准确匹配，我们需要区分以下三个概念：

1.  **产品型号 (Product Model)**：指您购买到的最终产品的销售型号（通常标注在外壳或包装上，例如 `FC6820`）。
2.  **主板型号 (Board Model)**：指设备内部核心主板或芯片组的型号。注意：**一个主板型号可能被用于多个不同的产品型号中**。
3.  **DejaOS 开发设备型号 (Development Device Model)**：指在 VSCode 插件中开发项目时选择的型号。它通常由 **主板型号 + 版本号** 组成（例如 `DW200_V20`）。
