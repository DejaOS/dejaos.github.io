# DejaOS 2.1.0

**DejaOS 2.1.0** 相比 2.0.0 重点提升了系统管理便利性和开发体验。系统管理应用的后续增强（退出界面、被动进入、生产日志下载）请参阅 [v2.1.5](./v2.1.5)。

### 1. 系统管理方式：可视化升级

此版本引入了全新的 [**System Manager App (系统管理应用)**](/docs/basics/managerapp)，彻底替代了 2.0.0 中的安全模式。

- **2.0.0 (旧方式)**：依赖 **安全模式**，需通过 RS485 串口线连接，操作繁琐且传输慢。
- **2.1.0 (新方式)**：支持 **可视化屏幕操作** 和 **Web 远程管理**。无需特殊线缆，直接在设备屏幕或电脑浏览器上即可完成应用安装、模式切换等操作。

### 2. 开发体验：速度优化

- **优化了 USB 数据传输协议**：解决了 2.0.0 版本在开发模式下代码同步较慢的问题，大幅提升了调试效率。
