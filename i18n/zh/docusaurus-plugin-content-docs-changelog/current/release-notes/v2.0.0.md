# DejaOS 2.0.0

DejaOS 2.0.0 是一个里程碑版本，确立了现代 DejaOS 的应用架构标准。相比于 1.0.0，它在规范性和底层管理上做了重大改进。

> **升级建议**：虽然 2.0.0 版本功能完整，但由于开发体验和管理方式相对落后，建议尽快升级至 2.1.0 版本。

### 1. 维护机制：新增安全模式

- **1.0.0**：缺乏底层的应急管理机制。
- **2.0.0**：新增 **安全模式 (Safe Mode)**，支持通过串口工具进行系统维护和应用安装。

### 2. 应用规范：全面升级

- **安装包格式**：由 `.zip` 升级为 `.dpk` (DejaOS Package)。
  - _注：支持 VSCode 插件一键打包（默认全量），替代了 1.0.0 的手动压缩。_
- **目录结构**：引入了更规范的层级，源码路径增加了 `/app/code` 前缀（如 `/app/code/src/main.js`）。
- **OTA 路径**：升级包放置路径简化为 `/upgrades.zip`（1.0.0 为 `/app/data/upgrades/APP_1_0.zip`）。
