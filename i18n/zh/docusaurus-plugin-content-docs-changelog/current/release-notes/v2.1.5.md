# DejaOS 2.1.5（最新推荐）

**DejaOS 2.1.5** 基于 **2.1.0**，是当前推荐的[系统管理应用](/docs/basics/managerapp)体验版本。

主要更新：

- **退出应用界面更清晰**：退出后显示明确状态页，而不再停留在上一应用的最后一页，避免误以为应用仍在运行。
- **被动进入**：主应用反复崩溃、看门狗检测到**连续重启 6 次**时触发熔断保护，设备停留在系统管理应用，便于排查「不失联」。
- **生产环境日志下载**：可在 Web 控制台的 **Maintenance & Logs** 中下载应用运行日志（生产环境无法像开发模式那样在 VS Code 实时看日志时尤其有用）。

详细说明与操作截图请参阅[系统管理应用](/docs/basics/managerapp)。
